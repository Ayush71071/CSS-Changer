CSS Changer


A lightweight, interactive web application that allows users to manipulate image properties in real-time using CSS variables and JavaScript. 


🚀 Features
This project demonstrates the power of updating CSS Variables dynamically. Users can adjust the following attributes: 
Width: Scale the image size from 0% to 100%. 

Border-Radius: Smooth out the corners of the image.

Base Color: Change the theme color and image background using a color picker. 

Padding: Adjust the spacing between the image and its colorful border.

Blur: Apply a Gaussian blur filter to the image.


🛠️ Technologies
UsedHTML5: Structured with semantic elements and range inputs. 

CSS3: Utilizes custom properties (variables), Flexbox, and Google Fonts ("Comfortaa").

JavaScript: Handles real-time DOM updates to CSS variables.  

Project Structure

├── index.html      # Main markup
├── css/
│   └── style.css   # Styling and CSS variables
└── js/
    └── script.js  # Logic for property updates

How to Use

Clone the repository to your local machine.

Open index.html in any modern web browser.  

Use the sliders and color picker in the controller section to see changes applied instantly to the city image.
